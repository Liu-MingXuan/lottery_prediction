# 彩票预测系统
此系统是自动爬取体彩大乐透、福彩双色球等，前期只预测大乐透和双色球。
# 项目开发流程
## 1 总体框架
### 1.1 底层选型
本项目使用python开发，数据库使用sqlite
### 1.2 预测方法
通过爬取历史的开奖记录，统计、分析、预测下次开奖哪些号码的概率最高，从而提供推荐购买的彩票号码。不要使用深度学习、机器学习，要从概率上预测概率，提供每个数的概率，同时输出一组概率最大的中奖号码预测组合。
### 1.3 开发方式
1. 首先爬取历史的中奖记录存入sqlite，注意要有时间、中奖号码，如果涉及前区、后区，要有字段体现，同时为了避免每次启动都要全部爬取一次，支持启动后只爬取sqlite中没有的记录
2. 根据历史的中奖号码，通过概率统计上的方法，得到预测每组号码的概率，例如 
{
    06:
        {
        前区:0.02,后区：0.002
        },
    07:
        {
        前区:0.022,后区：0.0031
        },
}
3. 最后输出一组最大概率的大乐透、双色球号码
例如：
{
    大乐透:{ 
        "前区":"--",
        "24":0.002,

    }，
